Introduction

My name is Patrick Mueller.

I work at IBM in Research Triangle Park, North Carolina on mobile web stuff.

This presentation is for you, so if you have a question, please interrupt me.

Questions for you - raise your hand:

  • I assume you are familiar with HTML, CSS and JavaScript;

  • How many people are familiar with the FireBug web debugger?

  • How many people are familiar with Safari's Web Inspector or Chrome Developer Tools; same as Safari's Web Inspector, different name

For folks familiar with Web Inspector / Chrome Developer tools, weinre reuses their user interface, so it should look familiar

For folks familiar with FireBug, weinre has similar capabilities, but a different user interface.
