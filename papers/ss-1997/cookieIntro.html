  • Supported by Netscape 2.0+ and IE.
  • Allow client specific data to be stored on the user's machine, and sent back to the server.
  • Used to persist user information
Caveats
  • Only supported on Netscape and Microsoft
  • Not supported by all servers
  • Only strings can be persisted
  • Amount of data to be persisted is limited
  • Some users specifically turn OFF cookie support
