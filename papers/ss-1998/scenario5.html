Scenario - Dial Phone via Voice
  • User navigates to Phone Dialer GUI
  • User presses steering wheel button to say 'Dial Phone'
  • User presses steering wheel button to enter numbers
  • As numbers recognized, sent to Java to redisplay on GUI
  • When user says 'Send', Java sends back command to Smalltalk to dial phone
  • Reset Audio to switch to phone
  • Send Class 2 message to dial phone
  • When phone call completed, restore audio from phone
