What the Car can do
  • DirectTV for passengers, DirectPC for internet, via antenna embedded in roof
  • Display for each passenger
    • Browse the web
    • Watch DirectTV
  • Console display for driver (no TV or Web Browser!)
  • Head up display on windshield
  • Configurable dashboard
  • CD/AM/FM/Internet Radio (Real Audio)
  • Mail, Stocks, Sports information
  • Voice command for driver
  • Text-To-Speech (TTS)
  • Accessing data from WorkPad (PalmPilot)
  • Access to vehicle status, doors open, seats occupied
  • Web access; car can be monitored and controlled via a Java applet running in a browser
