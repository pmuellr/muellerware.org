Head Up Display Component
  • Write-only interface
  • Actually controlled both the Head Up and Head Down displays (HUD and HDD)
  • Provided interface to all display updates on the Head Up and Head Down displays
    • road speed
    • engine speed
    • engine status
    • door lock/ajar
    • functions for the steering wheel buttons
    • indication of waiting e-mail
    • navigation information
    • microphone on/off
    • text message area for giving feedback to the driver
  • Used Application Variable patterns for single access point to the HUD
  • Used Actor pattern to asynchronously send commands
