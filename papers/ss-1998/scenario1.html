Scenario - Voice command to open the doors
  • User presses microphone button on the steering wheel
  • IPCM indicates microphone button is pressed
  • TTS stopped, if playing
  • Synchronous message to Java to stop RealAudio
  • Lower audio volume
  • Start voice recognition
  • Voice recognition recognizes 'Open All Doors'
  • Restore audio volume
  • Send Class2 message to open the doors
  • IPCM indicates doors open
  • Send message to web site indicating the doors are open
