Using Simulators
  • Built simulators so the entire system could be tested on a Thinkpad at home
  • Less travel required to visit actual hardware
  • Simulators built for:
    • IPCM - provided a dialog to simulate seat occupancy, etc
    • MML - provided a text area to display incoming CD track status, 'remote control' to change amplifier settings and command the CD player
    • Class 2 - provided incoming 'garbage' data, as well as a way to trigger engine sensor failure
    • HUD/HDD - Textual simulation of information displayed on HUD and HDD
