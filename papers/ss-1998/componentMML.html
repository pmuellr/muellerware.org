Mobile Media Link Component
  • Read-write interface
  • Packets sent as needed.
  • Provided interface to:
    • Amplifier
      • Volume / Balance / Fade
      • Audio source - Computer / CD Player / Phone
    • CD Player
      • Commands - Play / Stop / Next Track / etc
      • Track status - Current Disc / Current Track / Time Playing
    • Seat occupancy
    • Door ajar
  • Used Reading Serial Ports Design Pattern
  • Used Application Variable patterns for single access point to the MML
  • Used Actor pattern to asynchronously send commands
