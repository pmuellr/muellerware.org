• A client/server protocol to pass a CGI request from one program to another, and receive the results of the CGI request.

  • Current implementation uses tcp/ip for the transport, but other transports are possible (and advantageous)

    • Named pipes

    • FIFOs

    • Shared memory

  • CGI-Link client is actually a CGI-BIN program. CGI-Link server has been implemented in Smalltalk
