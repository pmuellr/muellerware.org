ncSeasMap

The program linked to below creates a .kml file suitable for use with Google Earth or Google Maps, which contains markers and data for NOAA sea / wave height forecasts for the coast of North Carolina. The .kml file itself is updated every few hours, to keep the data current.

The weather data is pulled from the NOAA; they provide lots and lots of data, so much so that it can be hard to find what you need. Sea/wave forecasts for the US are available by drilling down through the Marine Forecasts site.

ncSeasMap.kml - the .kml file to use with Google Earth, or view the .kml file in Google Maps.

ncSeasMap.rb - Ruby program to build the .kml file
